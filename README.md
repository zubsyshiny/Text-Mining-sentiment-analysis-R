# Lyrical Sentiment Analysis: Beyoncé vs. Taylor Swift
##*This repository contains a comprehensive data analysis project completed as part of my graduate coursework at Johns Hopkins University (JHU).

This project explores the evolution of lyrical themes and sentiment in popular music, using the #TidyTuesday dataset. It demonstrates my ability to handle non-structured text data and perform advanced visualizations in R.

## 🚀 Key Data Science Skills
- **Text Processing**: Tokenization and sentiment scoring using `tidytext`.
- **Dynamic Reporting**: Implemented via **R Markdown** for reproducible analysis.
- **Data Visualization**: Complex `ggplot2` implementations including faceted plots and sentiment trend analysis.

## 📊 Project Highlights
- **Sentiment Tracking**: Analyzing how album sentiments evolve over a decade-long career.
- **Correlation Analysis**: Investigating the relationship between lyrical complexity and commercial success (album sales).

## 🛠 Tools Used
- **Language**: R
- **Packages**: `tidyverse`, `tidytext`, `ggplot2`, `tidytuesdayR`
